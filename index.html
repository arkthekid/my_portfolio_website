Threads App

Project 1 (Independent Project)

This is a Threads clone app I developed using TypeScript and Next.js. I wanted to create a platform that mirrors the core features of a modern social media site, focusing on thread creation and user interaction. The app allows users to create threads, manage their profiles, and see real-time updates, making the experience dynamic and engaging. For the backend, I chose MongoDB to ensure robust and scalable data storage. Security was a top priority, so I implemented secure user authentication to keep user data safe. The interface, built with Next.js, leverages server-side rendering for efficient data fetching and smooth real-time updates. The goal was to create an app that feels fully responsive, providing a great user experience from front to back.

Tic-Tac-Toe Game

Project 2 (Independent Project)

This is a Tic-Tac-Toe game I coded using Python. I already knew the concept behind the game and wanted to create a program that had a good user interface and a bot that would be hard to beat. You can play against three types of opponents: another player, an easy bot, and a hard bot. Playing with another player is just taking your input again. The easy bot randomizes its moves. The hard bot uses the minimax algorithm to find the best possible move. This algorithm is a recursive algorithm that simulates every possible move. On the bot's turn, it picks the move with the highest score, and on your simulated turn, it picks the move with the lowest score. The user interface was built using PyQt5. Weather App

Project 3 (Independent Project)
